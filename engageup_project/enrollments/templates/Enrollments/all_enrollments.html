    help_outline ユーザーの削除・復元方法
    delete_outline 検定を削除する場合
    1. 削除したい検定の行をクリック、または左側のチェックボックスを選択します。
    2. 画面下部に黒い アクションバー が表示されます。
    3. 「一括削除」 ボタンを押すと、検定はゴミ箱へ移動します。
    settings_backup_restore 検定を復元する場合
    1. 画面上部の 「削除済みを表示」 ボタンを押して、ゴミ箱を表示します。
    2. 復元したい検定を選択します。
    3. 「一括復元」 ボタンを押すと、通常の一覧に戻ります。
    visibility 検定を公開・非公開にする場合
    1. 検定一覧の右側にある 公開 / 非公開スイッチ を切り替えます。
    2. 非公開にすると、検定は 一覧には残ったまま、受験者からは表示されなくなります。

      3. ※行選択をすることで・非公開を切り替えることができます。

    公開 / 非公開 と 削除 / 復元 の違い

    • 公開 / 非公開：「今はメンテナンス中だから一時的に隠したい」などの運営のコントロール
    • 削除 / 復元：「該当年度のモノはもう不要だから消したい」などのデータ整理・断捨離
